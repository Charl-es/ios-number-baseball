### 숫자야구 프로젝트 저장소

- 이 저장소를 자신의 저장소로 fork하여 프로젝트를 진행합니다

1. 제목: EeE-e 숫자야구게임 ⚾️

2. 소개: 플레이어에게 1부터 9까지의 한 자리 숫자 중복없이 3개를 입력받아 지정된 숫자 3개와
        자리와 숫자가 같으면 스트라이크, 동일 숫자가 포함되어 있으면 볼로 나타났을 때,
        9번 시도 중에 3 스트라이크면 사용자가 승리하고, 아니면 컴퓨터가 승리하는 게임
